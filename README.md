# Football Player Performance, Potential & Market Insights (Tableau)
## Overview
Interactive Tableau dashboard analysing player performance, potential, and market value to support scouting and recruitment decisions.
## Key Insights
- Nationality distribution for scouting focus
- Age vs overall performance patterns
- Potential vs overall to spot high-upside players
- Market value by club to understand investment patterns
