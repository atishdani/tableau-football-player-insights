# Football Player Performance, Potential & Market Insights

## Tools
Tableau, Data Visualisation, Data Analysis

## Description
Created an interactive Tableau dashboard analysing 18,000+ football player records to evaluate performance, potential, and market value trends.

## Key Features
- Player performance analysis
- Market value comparison
- Club & nationality insights
- Age vs potential analysis
- Interactive filters

## Insights
- Identified peak performance age range
- Found high-potential players
- Compared market values across clubs

## Skills Used
Tableau, Dashboard Development, Data Analysis, Data Visualisation
